# 🍽️ Restaurant Sales Performance Analysis

### Microsoft Excel | Business Intelligence | Data Analytics

A comprehensive sales performance analysis project built using Microsoft Excel to transform raw transactional data into meaningful business insights.

The project focuses on evaluating year-on-year sales performance, restaurant-level performance, weekly sales trends, and actionable business recommendations through data analysis and visualization.


## 📌 Project Highlights

- 📊 Analysed **774,075** sales transactions
- 🏪 Compared performance across **4 restaurant locations**
- 📅 Performed Year-on-Year and Weekly Trend Analysis
- 📈 Built professional Pivot Tables and Charts
- 💡 Generated business insights and recommendations



## 📖 Project Overview

This project analyzes restaurant sales data to identify performance trends, compare business growth across multiple locations, and provide actionable recommendations based on data-driven insights.

The analysis was performed using Microsoft Excel, leveraging Pivot Tables, Pivot Charts, lookup functions, and business reporting techniques to present findings in a clear and decision-oriented format.



## 🎯 Business Problem

Businesses generate large volumes of transactional sales data every day, but raw data alone does not support decision-making.

The objective of this project was to analyze historical sales data to answer key business questions, including:

- How did sales perform compared to the previous year?
- Which restaurant generated the highest revenue?
- How did weekly sales fluctuate over time?
- What business actions could improve future performance?
